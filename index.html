About the Project

This CubeSat prototype is designed for educational use, providing hands-on experience with satellite technology and space systems.

  • Real-time data collection and analysis
  • Advanced wireless communication system
  • Environmental monitoring capabilities
  • Integrated sensor array for precise measurements

Core Components

ESP32

Main processor handling data acquisition and wireless transmission with integrated Wi-Fi capabilities.

Arduino Mega

Secondary processor for LED visualization and data reception, with extensive I/O capabilities.

BMP280

High-precision environmental sensor for atmospheric pressure and temperature measurements.

MPU6050

6-axis motion tracking device combining accelerometer and gyroscope for precise orientation data.

NEO-6M GPS

High-performance positioning module providing accurate location and timing data.

OV7670 Camera

Low-power VGA camera module for image capture and visual data processing.

Operations

Data Collection

Advanced sensor array continuously monitors environmental parameters and satellite orientation in real-time.

Image Processing

Captures and processes visual data using advanced algorithms for enhanced image quality and analysis.

Wireless Transmission

Utilizes secure protocols for reliable data transmission and remote command reception.

LED Visualization

Dynamic LED display system provides real-time status updates and system diagnostics.
